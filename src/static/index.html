Instructions:
  1. Select a set of traces from the list on the left-hand side
  2. A table of values will appear in the middle. Each column corresponds to one variable.
  3. In the text box at the bottom of the middle column, try entering an "invariant" and pressing Enter.
  4. An invariant is any valid expression involving the variables that appear in the middle, numbers, arithmetic operations and operators and parenthesis. An invariant MUST evaluate to true or false.
  5. If you enter an invalid expression, an expression invovling variables that are not present, or an expression that does not evaluate to true or false, you will get an error message.
  6. Any row, for which the entered invariant holds, will be highlighted green after you press Enter. Any row for which the invariant does not hold will be highlighted red.
